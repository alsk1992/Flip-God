# Lister

Create, optimize, and manage listings on eBay, Amazon, and other platforms.

## Commands

| Command | Description |
|---------|-------------|
| `create listing on <platform>` | Create a new listing |
| `optimize listing <id>` | AI-optimize title and description |
| `update price <id> <price>` | Change listing price |
| `pause listing <id>` | Pause a listing |
| `resume listing <id>` | Resume a paused listing |
| `bulk list` | Create listings from top opportunities |

## Examples

- "create an eBay listing for product X at $29.99"
- "optimize my listing titles for better SEO"
- "update price for listing abc123 to $24.99"
- "bulk list the top 5 opportunities"

## Tips

- Use the optimizer to improve listing titles for search visibility
- Price slightly below competitors for faster sales
- Always include high-quality images
